# AESMS

This application is a student school project app about encryption.
The app is an alternative sms client that support encryption using AES passphrases.
It is intended to be used as-is.

No data is collected, not even usage statistics.

To enable encryption, click on the small lock next to the typing zone.
Then choose a passphrase to use with your contact.
When the lock is on, messages are encrypted.
If both contacts uses the sames passphrases, the crypted messages will be decrypted on display.

Happy SMS
